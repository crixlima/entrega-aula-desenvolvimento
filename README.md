# entrega-aula-desenvolvimento

Essa é um API de clinica medica para visualizar os medicos cadastrados e sua especialidade.

Ações funcionando:
*Criar cadastro - ok
*Remover Cadastro - ok
*Alterar cadastro - ok
*Buscar cadastro por ID - ok
*Buscar cadastro por nome - X

Foi utilizado as tecnologias
*JavaScript
*Express
*Postman
